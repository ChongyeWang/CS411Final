Movie Match

Welcome to Movie Match, an application created for the class CS 411: Database Systems in spring 2020 at UIUC. Here we put our knowledge of databases learned during class to the test, by developing an application using both a SQL and NoSQL database. Movie Match uses IMDB data collected from Kaggle, which is then stored in a Neo4j database. We used the SQL Database to store user information and movie reviews. Please use the Register button to create an account or the login button if you already have one. Please review the features list below to understand what you can do with our application. We hope you enjoy using our website!

Users Have Access To The Following Features:

  • Create an account using an email and password
  • Search our extensive movie database for information about different movies
  • Leave a rating and review for a movie
  • View users' profile pages and see reviews and statistics
  • Friend other users
  • See degrees of separation between users
