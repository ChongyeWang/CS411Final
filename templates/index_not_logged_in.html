About Movie Match:




Hello! Welcome to Movie Match, an application created for the class CS 411: Database Systems in the Spring Semester of 2020. Here we put our knowledge of databases learned during class to the test, by developing an application using both a SQL and NoSQL database. Movie Match uses IMDB data collected from Kaggle, which is then stored in a Neo4j database. We used the SQL Database to store user information such as emails and user friends.Please use the Register button located to the top right on the toolbar to create an account or the login buttion if you already have an account. Please review the features list below to understand what you can do with our application. We hope you enjoy using our website!




Users Have Access To The Following Features:




  • The ability to create an account using an email and password
  • The ability to search our extensive movie database for information about said movie
  • The ability to leave a rating for a movie
  • The ability a insert, update, and delete comments about a movie
  • The ability to view your profile page, which shows the amount of comments you've made and the average of your reviews
  • The ability to become friends with other users based on degrees of seperation
